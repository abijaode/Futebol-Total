# ⚽ Futebol Total

Bem-vindo ao repositório do **Futebol Total**, uma landing page responsiva criada para ser o maior portal de futebol da internet! Este projeto foi desenvolvido para fins educacionais, com foco em estruturação semântica e estilização de layouts para web.

## 📖 Sobre o Projeto

O **Futebol Total** simula a página inicial de um site esportivo onde os torcedores podem acompanhar as últimas notícias, conferir a tabela de jogos dos campeonatos (Série A e B) e interagir em um fórum dedicado. O objetivo principal do projeto foi aplicar na prática conceitos fundamentais de desenvolvimento front-end, criação de layouts responsivos e controle de versão.

## ✨ Funcionalidades

- **Design Responsivo:** A interface se adapta perfeitamente a diferentes tamanhos de tela (smartphones, tablets e desktops) utilizando CSS Media Queries e Flexbox.
- **Navegação Interativa:** Cards de conteúdo com links interativos que abrem em novas guias (`target="_blank"`), melhorando a experiência do usuário sem tirá-lo da página principal.
- **Tratamento de Imagens:** Utilização de propriedades CSS avançadas (`object-fit` e `object-position`) para garantir que as imagens dos cards mantenham a proporção e o foco correto, independentemente da resolução.
- **Seção de Newsletter:** Formulário estilizado para captação de leads e envio de novidades.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização, layout (Flexbox) e responsividade (Media Queries).
- **Git:** Controle de versão e registro de progresso do código.

## 🚀 Como executar o projeto

Como é um projeto puramente estático (Front-end), não é necessária nenhuma instalação complexa.
